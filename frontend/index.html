Willkommen auf unserer Website zur Leistungsbeurteilung 2 des Moduls 290


Auf dieser Website finden sie unter Login ein Formular. Dieses Formular ist mit einer Datenbank verbunden. Die Angaben, die sie dort eingeben werden validiert und in die Datenbank eingetragen und sie erhalten auf dem Bildschrim eine Nachricht, die ihnen die erfolgreiche Eingabe bestätigt. Falls eine Eingabe nicht der Validierung entspricht erhalten sie eine Fehlermeldung auf dem Bildschirm.
